Technical Information

This extension is built with React, Vite, and Tailwind CSS. It uses Chrome's Manifest V3 format and stores notes using chrome.storage.local.

The extension includes a content script that injects a sidebar interface, a background service worker for lifecycle management, and a popup interface for quick access to your notes.

Features to Test

  • Text Selection: Select any text on this page
  • Add to Notes: Click the "+" icon that appears
  • Sidebar: Press Escape or click the extension icon to open sidebar
  • Edit Notes: Modify the note content in the sidebar
  • Edit Titles: Change the note title from the default page title
  • Context Menu: Right-click selected text for quick access
